User Endpoints

Get All Users
GET /usuarios

Fetches a list of all users.

Create a New User
POST /usuarios

Creates a new user.

Request Body:

{
    "username": "exampleUser",
    "email": "user@example.com",
    "password": "securepassword",
    "plan": "free"
}
Get User by ID
GET /usuarios/:id

Fetches a user by their ID.

Update User by ID
PUT /usuarios/:id

Updates a user's details by their ID.

Request Body:

{
    "username": "newUsername",
    "email": "newemail@example.com",
    "password": "newpassword",
    "plan":"premium"
}
Delete User by ID
DELETE /usuarios/:id

Deletes a user by their ID.

Cocktail Endpoints

Get All Cocktails
GET /cocktails

Fetches a list of all cocktails.

Create a New Cocktail
POST /cocktails

Creates a new cocktail.

Request Body:

{
    "name": "Mojito",
    "category": "Classic",
    "season": "Summer",
    "photo": "url-to-photo",
    "taste": "Refreshing",
    "glassware": "Highball",
    "method": "Muddle, Mix",
    "ingredients": ["Mint", "Sugar", "Lime", "Rum", "Soda"],
    "instructions": "Muddle mint and sugar, add lime juice and rum, top with soda.",
    "userId": "user-uuid"
}
Get Cocktail by ID
GET /cocktails/:id

Fetches a cocktail by its ID.

Update Cocktail by ID
PUT /cocktails/:id

Updates a cocktail's details by its ID.

Request Body:

{
    "name": "Updated Name",
    "category": "Updated Category",
    "season": "Updated Season",
    "photo": "updated-url-to-photo",
    "taste": "Updated Taste",
    "glassware": "Updated Glassware",
    "method": "Updated Method",
    "ingredients": ["Updated", "Ingredients"],
    "instructions": "Updated instructions.",
    "userId": "user-uuid"
}
Delete Cocktail by ID
DELETE /cocktails/:id

Deletes a cocktail by its ID.
